# Variables — AI's most-reached-for tool

When you tell Cursor *track the user's score*, *store the title*, *remember the
last result*, the thing it reaches for first is a **variable** — a name that
holds a value.

You won't write many variables yourself. But you'll read AI-generated code that
uses them constantly. This first lesson is about recognizing them on sight.
